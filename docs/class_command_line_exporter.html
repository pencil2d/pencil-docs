Exports a Pencil2D file according to the specified options.

Parameters
inputPathPath of the file to export
outputPathsAt least one export destination file path
cameraName of the camera layer to use, or an empty string to select one automatically
widthWidth to use for export, or a negative value to use the dimensions from the camera layer
heightHeight to use for export, or a negative value to use the dimensions from the camera layer
startFrameFirst frame to include in the export(s)
endFrameLast frame to include in the export(s) or -1 to use the last keyframe or -2 to use the last keyframe including sound clips
transparencyWhether to export with transparency
Returns
true if the export was successful
