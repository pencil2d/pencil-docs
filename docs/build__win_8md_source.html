1 Building Pencil2D on Windows {#build_windows}
2 ============================
3 
4 These are instructions for building Pencil2D on a Windows PC. If you are using Mac go [here](@ref build_macos), and Linux please go [here](@ref build_linux).
5 
6 This guide is primarily targeted towards developers. If you just want to use the latest version, download it from our [nightly builds](https://www.pencil2d.org/download/#nightlybuild). This tutorial was made with Windows 10 in mind, however this will work with Windows 7 and up.
7 
8 There are 3 steps in total:
9 
10 1. Install %Qt SDK
11 2. Get the source code
12 3. Configure and compile Pencil2D
13 
14 ## TL;DR
15 
16 If you are an *experienced %Qt developer*, compiling Pencil2D would be extremely easy for you. Just open up `pencil2d.pro` in %Qt Creator and compile, that's it.
17 
18 ## Install %Qt SDK
19 
20 Pencil2D is built upon %Qt, you have to install it before you can compile the program.
21 
22 ### Official %Qt Installer (Recommended)
23 
24 - Download the <b>%Qt Online Installer for Windows</b> from [%Qt Downloads](https://www.qt.io/download-open-source/)
25 - Open up the installer, you can skip the step of creating %Qt account, it's not necessary.
26 - In the next step, choose the %Qt version and your C++ compiler.
27  - If you have no idea what to do, please select latest `%Qt 5.9.x -> MinGW 5.x` and `Tools -> MinGW 5.x`.
28 - Agree to the license and start the installation. It will take a long time to download all of the files, so be patient. When the installation is complete, press `Done` and it will launch %Qt Creator for you.
29 
30 ## Get Source Code
31 
32 - You can simply download the source code archive [here](https://github.com/pencil2d/pencil/archive/master.zip).
33 - Or get the source via [Git](https://github.com/pencil2d/pencil.git) if you plan to contribute to the Pencil2D project.
34 
35 ## Building the application
36 
37 Now it's time to build the application.
38 
39 ### With %Qt Creator (recommended)
40 
41 - Open up the %Qt Creator application.
42 - From the menu bar select **File** and then **Open File or Project**. Navigate to Pencil2D's root source folder and open the `pencil2d.pro` file.
43 - Next, you'll be asked to configure your `kits` for the project. Kits determine compilers, target environment, and various build settings among other things. The Desktop option should be the only one checked. Click Configure Project to complete the kit selection.
44 - Now all you have to do to is click the plain **green arrow** in the bottom left corner or press `Ctrl+r`. A small progress bar will show up on the bottom right and console output will appear in the bottom section.
45 - If everything goes well then the Pencil2D application that you build will open up automatically and you're done!
46 
47 If any error occurred, the issues tab will open up at the bottom and display error messages. Please search the [Pencil2D Issue tracker](https://github.com/pencil2d/pencil/issues) or create an issue If you can't find anything. Be sure to include as much detail as you can in your report!
48 
49 ### With Visual Studio
50 
51 Yes, you can compile Pencil2D with Visual Studio if you like.
52 
53 - Firstly, have Visual Studio 2015 or later versions installed.
54 - Next, install the %Qt SDK which matches your VS version (e.g. %Qt 5.6 msvc2015 64bit if you use VS2015).
55 - Download and install the [%Qt Visual Studio Add-in](http://doc.qt.io/archives/vs-addin/index.html).
56 - Open Visual Studio, from the Menu bar -> %Qt VS Tools -> Open %Qt Project File (.pro), navigate to the Pencil2D source folder and select `pencil2d.pro`.
57

59
